## 📋 Project Overview
This is a hardware-based robot car project that uses an IR proximity sensor to detect obstacles. The system is built using a purely analog circuit

*Note: This repository contains only images and documentation of the build, as there is no software code involved.*

## 🛠 Components Used
- IR Proximity Sensor
- L293D / L293NE Motor Driver IC
- BO Motors
- 7805 Voltage Regulator
- 9V Battery & Switch
- Zero PCB, Pin Headers, and Wires
- Custom Wooden Frame (9x4cm & 9x2.5cm) and Wheels

## ⚙️ How It Works
The IR sensor detects nearby objects and sends a signal to the L293D motor driver IC. Based on the sensor's output, the IC controls the direction and movement of the BO motors, allowing the car to react to its surroundings without any programmable logic.
